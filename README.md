Terminal-based ToDo list

Hello everyone!
This is my first app, I have ever built. It is simple ToDo list, that has full sync with ToDoist app,
launchable from any Linux terminal (just type tdl). It uses simple keyboard like Vim, ideal for mouseless enviroments.
It isnt perfect, bu I tried my best. It still has some flaws, but I will fix them, and work on new features actively.

installation:
Arch: aur package (for example yay tb-tdl)
installation from other distros will be added soon.

Config:
We automatically generate config in ~/.config/
Here you can customize colours, keybinds, and put here your API for ToDoist.

Future:
I plan to add much more features soon. Main future goal will be interface customisatios, and sync support for other ToDo list apps, not only ToDoist.
